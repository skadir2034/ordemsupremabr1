# Regras do Projeto / Project Rules

- **Firebase Security Rules:** Siga estritamente esta instrução: **NÃO altere** o arquivo `firestore.rules` ou as regras de segurança do Firebase de forma automática ou silenciosa. Se for necessário modificar qualquer regra de segurança no banco de dados para o funcionamento do app, **avise o usuário previamente** detalhando as alterações necessárias em vez de aplicá-las de forma automática.
